# 水下清洁机器人项目

![上海海洋大学logo](Docs/imgs/shou-logo.png)

## 项目概述

本项目旨在开发一款基于负压吸附技术的无线自主轮船清洁机器人，参考了现有负压式泳池机器人的设计理念，并针对轮船清洁的特殊需求进行了优化和创新。

### 主要功能与目标
- 实现对轮船船体表面的自主清洁
- 采用负压吸附技术确保机器人在船体表面的稳定附着
- 具备无线自主导航和定位能力
- 适用于各种水下环境的轮船清洁作业

### 背景与意义
随着航运业的发展，轮船船体表面的清洁维护成为一项重要且具有挑战性的任务。传统的人工清洁方式效率低下、成本高昂且存在安全风险。本项目开发的水下清洁机器人旨在解决这些问题，提高清洁效率，降低成本，并保障作业人员的安全。
